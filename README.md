## Getting Started

* Clone the repository with:

```bash
git clone https://github.com/RiteshPuvvada/Airdrop-with-Solana-web3.js.git

cd Airdrop-with-Solana-web3.js
```
* Install the dependencies:

```bash
npm install
```

* Testing functions

```bash
node index.js
```
